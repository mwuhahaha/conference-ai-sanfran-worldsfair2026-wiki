# Source Corpus

This folder holds the public and derived source corpus for the AI Engineer World's Fair 2026 wiki.

For agent navigation, start with:

- `../../AGENT.md`
- `../../AGENTS.md`
- `../../wiki/resources/agent-source-index.md`
- `../../wiki/resources/source-boundary.md`

Source layers:

- Official schedule and speaker mirrors are canonical for conference facts.
- YouTube metadata, captions, livestream transcripts, Whisper fallback transcripts, slide frames, reconstructed slide crops, and OCR are supporting evidence.
- Public company sites, product docs, and professional profiles can enrich people/company/topic pages when directly relevant and clearly labeled.

Do not add private recordings, unrelated local vault notes, diary material, or heavyweight temporary media caches to the publishable source corpus.
