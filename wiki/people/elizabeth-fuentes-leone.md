# Elizabeth Fuentes Leone

## Profile
Elizabeth Fuentes Leone is a Developer Advocate at [[amazon-web-services|Amazon Web Services]] who helps developers move AI applications and agents from initial code into production deployment, observability, debugging, and iterative improvement. Her two World’s Fair sessions connect that operational lifecycle with context engineering: one is a hands-on agent shipping workshop, while the other examines how [[agent-memory]] and [[mcp|Model Context Protocol]] can provide durable, structured context when a model’s context window is not sufficient.

## Biography
Elizabeth Fuentes Leone brings together data analytics, machine learning, developer education, open-source work, and live technical demonstration. At AI Engineer World’s Fair 2026, she appears first alongside Sandhya Subramani in the June 29 Track 9 workshop “Agent Speedrun,” which follows an agent through idea, coding, deployment, observation, repair, and release. On June 30, she presents “The Infinite Context Window Is a Myth” on Expo Stage 3 SW, shifting from the mechanics of shipping agents to the design of the context they receive. Together, the sessions frame production agent engineering as both an operational discipline and an information-management problem: developers must observe and fix running systems while deliberately selecting, storing, and exposing task-relevant context through mechanisms such as [[agent-memory]] and [[mcp|Model Context Protocol]].
