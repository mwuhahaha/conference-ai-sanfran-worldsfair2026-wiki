# Kevin Orellana

## Profile
Kevin Orellana is a software engineer at [[amazon-web-services|Amazon Web Services]] focused on the controlled execution infrastructure behind AI and coding agents. His work spans [[ai-sandboxes|AI sandboxes]], browser automation, and the runtime isolation needed for agents to execute code and interact with websites safely at scale.

- [LinkedIn](https://www.linkedin.com/in/kevinorellana/)
- [X / Twitter](https://x.com/KevssOrellana)

## Biography
At AWS, Orellana builds sandboxed code-execution and browser-automation systems for large populations of agent tasks. This work connects runtime isolation with the broader [[agent-security|agent security]] problem: agents need controlled access to files, dependencies, websites, data, and generated artifacts without inheriting unrestricted host privileges. He previously worked on Amazon Bedrock’s model-serving platform, where he tech-led the launch of Anthropic’s Claude Sonnet on Bedrock and contributed to infrastructure supporting several frontier-model launches. His World’s Fair session brings that platform experience to the operational question of what fails when LLMs write and run code across 1,000 sandboxed tasks.
