## Session Description
We are in the era of Software Factories, where the entire SDLC is being automated by agents. We will cover how we are approaching self-improving software factories leveraging dedicated agents to update skills, persistent cross-harness memory, and implementing feedback loops to ensure that software factories continually improve.

## Summary
Suraj Gupta's Expo Stage 3 SW session frames Warp's work as part of the World's Fair software-factories theme: AI-native development systems where coding, review, orchestration, and quality gates are increasingly handled by coordinated agents. The talk focuses on how Warp is thinking about self-improving agentic developer environments, including dedicated agents that update skills, persistent memory that carries lessons across harnesses, and feedback loops that let a software factory improve from its own outcomes rather than remaining a static automation pipeline.
